# KRunner Custom Command

## Installation

1. Install `python-dbus` and `translate-shell`.
2. Run `./install.sh`
3. Start `main.py`
4. You might want to set up autostart for `main.py` in your KDE settings.

## Usage

1. Open KRunner.
2. Type "trans text." to translate text. The text must end with a period (`.`) to trigger the translation.
